Wrench Pedagogic Modules

A guided set of educational activities focused on parallel computing, distributed computing, and cyberinfrastructure concepts through using the WRENCH simulation framework.

Goal

The goal is for students to achieve learning objectives in the areas of parallel computing, distributed computing, and cyberinfrastructure computing. This is achieved via pedagogic modules that explore, at first elementary and later sophisticated, relevant concepts. These modules include hands-on "activities" in which students explore and experiment with simulated application using the WRENCH simulation framework. These modules can be used to enhance/augment existing courses, such as university or on-line courses, or can simply be done in sequence by independent students.

About WRENCH

WRENCH is framework for simulating the execution of parallel and distributed applications on cyberinfrastructures. It is based on the lower-level SimGrid simulation framework, which provides the necessary accurate and scalable simulation abstractions.
